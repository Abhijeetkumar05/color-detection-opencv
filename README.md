# 🎨 Real-Time Multi Color Detection using OpenCV

A Python project that detects 10 colors in real-time using a webcam and generates graphs automatically.

## 🛠️ Technologies Used
- Python 3, OpenCV, NumPy, Matplotlib

## ✅ Features
- Detects 10 colors: Red, Green, Blue, Yellow, Orange, Pink, Purple, Brown, White, Black
- Draws bounding boxes with labels on detected colors
- Generates Bar Graph, Pie Chart, Timeline and Dashboard on pressing Q
- Saves results to CSV file

## ▶️ How to Run
pip install opencv-python matplotlib numpy
python color_detection.py
